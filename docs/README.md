# Documentação RXfin

## Estrutura

```
docs/
├── README.md           ← este arquivo
├── adr/                ← Architecture Decision Records
│   └── 001-monorepo.md
└── guides/             ← guias operacionais

```

## ADRs (Architecture Decision Records)

Toda decisão técnica significativa deve ser registrada como ADR.

Formato: `docs/adr/NNN-titulo.md`
